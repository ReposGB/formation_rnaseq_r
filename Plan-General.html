ghislain.bidaut@inserm.fr

Objectifs

Savoir mener une analyse bioinformatique de données de RNA-seq .

  • Dans l’environnement utilisé par les bioinformaticiens: à savoir R/Bioconductor.

  • Travailler de manière participative pour se familiariser avec les technologies utilisées en bioinformatique pour le traitement de données NGS et communiquer plus efficacement avec les bioinformaticiens!

  • Lors de cette formation, nous mettrons l’accent sur les technologies open-source mises au point et utilisées dans le monde de la recherche.

Présentation des participants

  • Dans quel domaine travaillez vous ?

  • Quelle est votre expérience en analyse de données bioinformatique et RNA-seq ?

  • Qu’attendez vous de cette formation ?

Au menu

  • Identifier les grandes étapes d’une analyse bioinformatique RNA-seq
  • S’approprier l’environnement de travail RStudio
  • Importer ses données sous RStudio
  • Manipuler des données matricielles sous R
  • Identifier les gènes différentiellement exprimés par le programme edgeR
  • Produire des figures publiables de qualité
    • Heatmaps par le programme ‘pHeatmap’
    • Volcano plots par le programme ‘ggplot2’
  • Automatiser son analyse RNA-seq à l’aide d’un script R.
